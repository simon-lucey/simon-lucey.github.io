# Simon Lucey academic site starter

This is a minimal static GitHub Pages site.

## Files you will likely edit

- `index.html` — home page text
- `admissions.html` — graduate admissions page text
- `publications.json` — featured publications
- `style.css` — styling
- `assets/headshot.jpg` — headshot image

## GitHub Pages setup

1. Create a GitHub repository, ideally `simonlucey.github.io`
2. Upload all files in this folder
3. In GitHub, go to **Settings → Pages**
4. Under **Build and deployment**, select **Deploy from a branch**
5. Choose **main** and **/(root)**
6. Save

Your site should then appear at your GitHub Pages URL.
